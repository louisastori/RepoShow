Apercu du projet

Mini CRM comble le vide entre les solutions d'entreprise complexes et les feuilles de calcul. Le projet presente une architecture capable de gerer des centaines de clients avec des temps de reponse inferieurs a la seconde.

01

Zero latence

Requetes SQLite optimisees et rendu concurrent React.

02

Donnees locales securisees

Stockage chiffre de la base de donnees avec acces local.

Comment installer

$ git clone https://github.com/dev/mini-crm.git
$ cd mini-crm
$ npm install
$ npm run dev
